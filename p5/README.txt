###- K MEANS CLUSTERING -###

The application does a clustering people according to their income and spend datas.

When the application is run, a data.txt file is read line by line.

First line content INCOME and SPEND column information, so first row is passed without any applying.

The app wants a K value from user to generate K different clustur.

Then it creates K different random value and plot in the Income(X label) and Spend(Y label) diagram with different colors.

In a while loop, colorPoints and newMean method is called until reach the candidate values optimal positions.
